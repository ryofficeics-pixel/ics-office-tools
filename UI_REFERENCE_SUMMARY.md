# UI Reference Summary

The dashboard follows the IC Solution visual direction from the available ICS UI reference:

- Off-white page background
- White card surfaces
- Soft grey borders
- Subtle elevation shadow
- Moderate rounded corners
- Strong black primary buttons
- Compact uppercase status pills
- Black rounded icon blocks
- Clear hierarchy with large headings and muted helper text
- Mobile-first card flow with no fixed card height
- Consistent spacing and readable tile layout

The implementation uses these patterns without importing any external UI library.
